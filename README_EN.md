## Feature Overview

- **Three-Role Orchestration**: PM Commander / Delivery Director / Technical Consultant — one skill pack covers the entire delivery team
- **Nine Atomic Capabilities**: Document generation, meeting minutes, risk radar, progress tracking, resource coordination, client communication, requirement analysis, acceptance & delivery, data collection
- **Structured Routing**: Dual-layer routing engine (keyword + regex + semantic) for precise intent matching
- **Externalized State**: Project state persisted as YAML files, shared across sessions and roles
- **China-Localized**: Yunzhijia / Feishu / DingTalk data collection, GB/T government document standards, Chinese delivery practices
- **Multi-Skill Orchestration**: Parallel merge, sequential execution, and conditional branching — three orchestration modes

## Target Users

- **Project Managers**: Weekly/daily report generation, meeting minutes, risk assessment, progress tracking
- **Delivery Directors**: Cross-project panoramic dashboard, resource scheduling, project health ranking
- **Technical Consultants**: Requirement analysis, solution review, GAP analysis, technical risk assessment

## Quick Start

1. Install this skill in QoderWork
2. On first use, type "help me write a weekly report" and the system will guide you through creating your first project
3. Once the project is created, all capabilities are available immediately

Example commands:
- "Help me write this week's report"
- "Organize today's meeting minutes"
- "Assess the project risks"
- "How is the project progressing?"
- "Write a status update email for the client"
- "Prepare materials for tomorrow's project review" (composite orchestration)
